# ChristoBuzz

Social media web app demo inspired by Facebook, Instagram, TikTok, Telegram.

## Features
- Posts, Stories, Reels, Chat, Marketplace
- Profile with posts
- Music player
- Notifications
- Ads and AI placeholders
- Supabase authentication and DB

## Setup
1. Replace Supabase URL and anon key in supabase.js.
2. Deploy to Vercel/Netlify or run a local server.
3. Open index.html.
