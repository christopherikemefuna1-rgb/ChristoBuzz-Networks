# ChristoBuzz Networks

ChristoBuzz is a social media platform combining features of:
- Facebook (posts & profiles)
- Instagram (media sharing)
- TikTok (short videos)
- Marketplace (coming later)

## Tech Stack (Planned)
- Frontend: HTML / CSS / JavaScript (later React)
- Backend: Supabase
- Hosting: Render or Netlify
